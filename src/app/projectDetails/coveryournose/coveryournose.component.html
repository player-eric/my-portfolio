Despite the significant vaccination process, people may still need to wear masks for a long time when they are outdoors. This web app CoverYourNose was developed to help people check if they are wearing masks properly.

Powered by an Single Shot MultiBox Detector (SSD) trained on ~10k annotated images, and a few HAAR-feature based faicial landmark detectors, CoverYourNose achieves satisfactory performance on detecting wrongly worn masks. For more details, please refer to this report and the code .

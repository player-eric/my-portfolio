An application transfers any image into Japanese Anime style with the help of community detection algorithm and neural style transfer algorithm.

2,000 Anime posters were collected and constructed into a network based on similarities. And then various community detection algorithms were run on this network to filter out the most representative posters of different genres. These posters are further used as style reference images in neural style transfer algorithm to stylize the input images.

This project won the best machine learning award in 2019 NUS SOC workshop, see our poster here.
